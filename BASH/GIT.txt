Push an existing folder
-----------------------

cd existing_folder
git init
[ git remote remove origin ]
git remote add origin https://gitlab.com/przemyslaw.kuznicki/cicd-tasks-app.git
git add .
git commit -m "Initial commit"
git push -u origin master

--------------------------------
